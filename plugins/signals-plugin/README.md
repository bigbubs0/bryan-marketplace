# Signals Plugin

Biotech signal tracking - funding, layoffs, clinical data, weekly digests, and client briefs.

## Commands

| Command | Description |
|---------|-------------|
| `/signals:intake [signal]` | Log a biotech signal to Notion with Company Core linking |
| `/signals:brief [company]` | Generate a client intelligence brief from Notion data |
| `/signals:weekly [timeframe]` | Weekly digest of all logged signals (defaults to past 7 days) |
