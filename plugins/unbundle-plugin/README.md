# Unbundle Plugin

Operationalize a "bundled" term - a soft skill (charisma, executive presence, active listening), a fuzzy workflow (client onboarding, content batch, weekly review), or a belief/identity (strategic, disciplined, curious) - into concrete, trainable, daily actions.

## Skill

Skill-only (no slash command). Auto-triggers when you say "unbundle this", "operationalize X", "what does X actually mean", "how do I actually become more Y", "break this workflow into steps", "what does a [role] actually do daily", or "turn this into a checklist".

Runs a three-track interview (Workflow / Soft Skill / Belief) via AskUserQuestion, then returns 5-12 daily-trainable actions, each with what-good-looks-like and where-to-do-it, plus an optional micro-routine.
