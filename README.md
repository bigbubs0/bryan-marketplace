# Bryan's Plugin Marketplace

Claude Code plugin marketplace for biotech recruiting operations.

## Plugins

| Plugin | Commands | Description |
|--------|----------|-------------|
| candidates | `/candidates:add` | Candidate intake with Notion auto-linking |
| signals | `/signals:intake`, `/signals:brief`, `/signals:weekly` | Biotech signal tracking and intelligence |
| outreach | `/outreach:draft` | Personalized outreach with Voice DNA profiles |

## Install

```
/plugin marketplace add bigbubs0/bryan-marketplace
/plugin install candidates@bryan-marketplace
/plugin install signals@bryan-marketplace
/plugin install outreach@bryan-marketplace
```
